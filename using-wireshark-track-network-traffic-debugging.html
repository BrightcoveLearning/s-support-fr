Présentation

Wireshark est une application open source populaire qui permet d'analyser les protocoles réseau de manière beaucoup plus approfondie que ce que vous pouvez obtenir, par exemple, via la partie Réseau des outils de développement de navigateur. Cela peut être très utile pour diagnostiquer les problèmes liés aux applications Internet telles que le lecteur Brightcove ou aux applications qui interagissent avec Brightcove via nos API.

Même si votre connaissance des réseaux n'est pas suffisamment approfondie pour bien comprendre les informations que vous obtenez de Wireshark, la capture des informations peut être d'une grande aide pour le support de Brightcove dans le diagnostic des problèmes, et nos représentants du support peuvent vous demander s'il est possible pour vous de capturer des données réseau à partir de Wireshark afin qu'ils puissent mieux vous aider.

Les étapes ci-dessous vous guideront tout au long de l'installation et de l'utilisation de base de Wireshark.

Installation

  1. Si Wireshark n'est pas déjà installé, rendez-vous sur https://www.wireshark.org/#download et téléchargez le programme d'installation correspondant à votre plate-forme.
  2. Exécutez le programme d'installation. Si des options vous sont proposées, installez tous les composants disponibles. (Les options varient selon la version et la plateforme.)

Utiliser Wireshark

Important: avant de commencer, il est préférable de fermer autant d'applications qui accèdent à Internet que possible (par exemple: clients de messagerie, applications de messagerie instantanée ou de réseaux sociaux, Dropbox, Evernote et autres applications de stockage basées sur le cloud). Cela permettra de minimiser le bruit-fr/ des transferts réseau qui ne sont pas liés au problème que vous essayez de diagnostiquer.

  1. Lancez Wireshark.
  2. Sélectionnez un filtre de capture - celui que vous souhaitez affichera probablement la plus grande partie ou au moins une bonne partie du trafic:
  3. Double-cliquez sur le filtre dont vous souhaitez commencer à capturer les données:
  4. Avec la capture de données en cours d'exécution, recréez le problème que vous essayez de diagnostiquer en chargeant le lecteur, en exécutant l'application ou quoi que ce soit d'autre.
  5. Une fois que vous avez terminé de reproduire le problème, cliquez sur le bouton Arrêter la capture de données dans Wireshark:
  6. Maintenant, exportez les données de Wireshark en cliquant sur le bouton Enregistrer le fichier de capture:
  7. Enregistrez le fichier au format par défaut sur votre ordinateur.
  8. Envoyez le fichier au support Brightcove dans une demande d'assistance détaillant le problème que vous rencontrez.
